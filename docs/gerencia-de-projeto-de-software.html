Ementa

  4. Modelos de qualidade de processo de gerência de projeto de software (4h).
  5. 5Áreas de conhecimento da gerência de projetos aplicadas a projetos de software (48h): escopo, tempo, custo, qualidade, recursos humanos, comunicações, riscos, aquisições, integração e partes interessadas.
